Las 4 sentencias mas usadas

Git Init:

Esta sentencia se utiliza para iniciar un nuevo repositorio Git en un directorio. Inicia el seguimiento de los cambios en los archivos en ese directorio y crea un repositorio local.

Git Clone:

Se utiliza para clonar un repositorio Git existente desde un servidor remoto (como GitHub, GitLab o Bitbucket) en tu máquina local.

Git add:

Mueve los cambios del directorio de trabajo al area del entorno de ensayo.

Git Commit:

Después de agregar los cambios al área de preparación con git add, puedes confirmar esos cambios en un nuevo "commit" con esta sentencia.
